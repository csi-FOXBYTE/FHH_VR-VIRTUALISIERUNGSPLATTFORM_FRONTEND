## 4. Administration

Der Administrationsbereich ermöglicht den Zugriff auf zentrale Systemverwaltungsfunktionen.

Funktionen:

- Hochladen und Organisieren von Basisdaten.
- Konfiguration von Systemeinstellungen (z. B. APIs, Anwendungen).
- Einsicht in sicherheitsrelevante Systemlogs.
- Verwaltung von Benutzern und Gruppen.

Schaltfläche: Administrationsbereich öffnen → Zugriff auf Verwaltungskonsole.

### 4.1 Datenverwaltung

- Upload und Organisation von Basisdaten.
- Definition von Sichtachsen (z. B. zur strukturierten Darstellung von Daten).
- Verwaltung bestehender Datenbestände.

Schaltfläche: Zur Datenverwaltung

#### 4.1.1 Ebenen

Die Ebenen beschreiben die Basisdaten.

Es besteht die Möglichkeit, bereits vorhandene 3D Tiles, Terrain oder Imagery einzubinden und direkt zu verlinken – ohne dass eine Umwandlung notwendig ist.

> Der Wandlungsknopf befindet sich neben Hinzufügen mit Klick auf den Pfeil nach unten.

![alt-text](/help/de/administration_data-management_overview.png)

Zusätzlich können Daten konvertiert werden, zum Beispiel:

- CityGML → 3D Tiles
- DGM-Format (Terrain) → Cesium Terrain Tiles

Dabei ist wichtig, dass die Daten immer in einer ZIP-Datei verpackt werden. Die interne Ordnerstruktur spielt dabei keine Rolle.

> Ein weiterer Punkt ist die Angabe des EPSG-Codes. Da manche CityGML-Formate diesen nicht enthalten, muss er zwingend definiert werden.

Nach dem Klick auf „Wandeln“ werden die Daten zunächst hochgeladen. Direkt im Anschluss startet der Umwandlungsprozess. Den Fortschritt können Sie jederzeit in der Tabelle einsehen. Sollte ein Fehler auftreten, wird dies in der Fortschrittsanzeige mit einem X kenntlich gemacht.

### 4.2 Konfiguration

- Verwaltung von Systemeinstellungen.
- Konfiguration von Daten, APIs und Anwendungen.
- Ermöglicht die technische Integration und Anpassung des Systems.

> Über auf und zuklappen der einzelnen Reiter lassen sich die verschiedensten Dinge konfigurieren (z.B. E-Mail Templates, Standard EPSG Code, ...)

> WICHTIG! 
> Um zu speichern verwenden Sie den speichern Knopf in der rechten oberen Ecke, nach drücken von Speichern kann nicht mehr auf den vorherigen Wert zurückgesetzt werden. Das geht nur wenn noch nicht gespeichert wurde!
